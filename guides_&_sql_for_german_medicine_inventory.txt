🔵 This will work on any numeric column except dates,
Just change the column name.⬇️

SELECT * 
FROM sales_business.german_medicine_inventory
ORDER BY retail_price ASC;

🔵 Find any medicine you think that is inside the table⬇️

SELECT * 
FROM sales_business.german_medicine_inventory
WHERE medicine_generic_name LIKE 'f%';


🟢Type the first letter of the name of the medicine you want to search for (beside the %). [With lowercase letters]
